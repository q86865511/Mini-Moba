# Asset Credits

All assets in this project are **original**, generated locally for this workspace
(procedural raster generation with Pillow). No third-party game IP is referenced.

## Packs

- **`moba_asset_pack/`** — static raster assets: a 2048×2048 three-lane map, sliced
  map tiles (128/256), and five 1024×1024 transparent hero PNGs.
- **`moba_interaction_pack/`** — animated sprite sheets (256×256 frames, 6 columns,
  states idle/move/attack/hurt/death) for heroes, lane minions, neutral monsters,
  towers and cores, plus per-sheet `*_actions.json` (frame rects, anchors, collision
  circles, hurtboxes, attack events) and `map/interactive_map_zones.json`.

License note (from the packs): "Original generic assets generated for this workspace;
no third-party game IP referenced."

Procedurally generated audio (move SFX, ambient music) is original to this project.
