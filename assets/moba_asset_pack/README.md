# MOBA Asset Pack

Original generic MOBA-style raster assets generated locally for this workspace.

## Files

- Map: `outputs/moba_asset_pack/map/moba_three_lane_map.png` (2048x2048 PNG)
- Preview: `outputs/moba_asset_pack/preview/moba_asset_pack_preview.png`
- Heroes: transparent 1024x1024 PNGs in `outputs/moba_asset_pack/heroes/`

## Hero Roster

- **Ember Vanguard** (fighter): `outputs/moba_asset_pack/heroes/ember_vanguard.png`
- **Frost Ranger** (marksman): `outputs/moba_asset_pack/heroes/frost_ranger.png`
- **Storm Oracle** (mage): `outputs/moba_asset_pack/heroes/storm_oracle.png`
- **Iron Sentinel** (tank support): `outputs/moba_asset_pack/heroes/iron_sentinel.png`
- **Night Veil** (assassin): `outputs/moba_asset_pack/heroes/night_veil.png`

## Generation Notes

- Style: stylized top-down/semi-chibi game assets.
- Constraints: original generic MOBA fantasy theme, no logos, no watermark, no copied game layout.
- Mode: local procedural raster generation with Pillow because the built-in image generation tool was not available in this session.
